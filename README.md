# Sistema de Misiones 

Aplicación **sin frameworks** (solo JavaScript, HTML y CSS) para administrar ofrendas de misiones en la iglesia por persona y por mes.

## Funcionalidades

- Login de administrador.
- Registro de personas con campos obligatorios:
  - Nombre
  - Teléfono
  - Monto prometido
- Abono inicial opcional.
- Edición de personas.
-Eliminación de personas
- Registro/edición mensual de abonos por persona.
- Exportación de PDF por persona con título **"Reporte de misiones"**, logo y tabla mensual.
- Diseño mobile-first, responsive y con animaciones.
